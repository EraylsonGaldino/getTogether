API para obter recomendações de clientes

Instruções:

A solicitação deve ter a seguinte estrutura: https://recomend.herokuapp.com/{cidade}/{palavras_chaves}

Por exemplo: https://recomend.herokuapp.com/Caruaru/comer, pizza

Para mais de uma palavra, é preciso separar por virgula.

O retorno terá o seguinte formato:

{"Twitter": "twitter", "User": "contaDoUsuario"}

Por exemplo:

{"Twitter": "Uma pizza grande só pra mim, o deliciaaaa "User": "pablowagner04"}
